What is this?

Poker Planner is a free and open-source project lets you make estimations with planning poker (or scrum poker) directly in Slack, without any need of external software. It can be a useful tool for agile remote teams. You can read more about planning poker on wiki. This service is free in goodwill, and hopefully will remain free. You can install the app with "Add to Slack" button above. If you want to host your own app, please check the setup guide on Github.

Usage

After successful installation, /pp slash command will be avaliable in your team workspace.

/pp some topic text
This command starts a poker planning session on specified topic, or simply anything you typed after /pp. It works in:

  • Public channels
  • Private channels
  • Conversations between multiple people (they are basically private channels)
This command will automatically find active (online and not-away) users in the current channel/group and add them as participants to poker planning session. Therefore /pp command does not work in direct messages.

WARNING: Topic text cannot start with "config" and "help" (case-sensitive). They have another functionalities which will be described below.

/pp some topic text @user1 @user2 ...
This command works exactly like above, however specifically mentioned users will be added to the session, even if they are offline. @channel, @here mentions are also supported. As described above, if there is no mention, Poker Planner will work as @here mentioned.

/pp config value1 value2 ...
This command lets you customize poker values just for your team. If you want to estimate with t-shirt sizing, just type /pp config XS S M L XL. After this command, all the voting sessions will have just these options, until you configure Poker Planner again with /pp config command.

At least 2 values must be provided, seperated with space character. Each value cannot be more than 20 characters, and maximum 25 poker values are supported.

/pp config reset
This command resets the poker values back to default settings.

/pp help
This commands prints a cheatsheet of usage.

Required Slack Permissions

  • commands - Add slash commands and add actions to messages (and view related content)
    • We will add /pp slash command
  • chat:write:bot - Send messages as Poker Planner
  • channels:read - Access information about user’s public channels
    • When /pp command is used in a public channel, we will get member list of it, in order to detect active users (to be added as participants).
  • groups:read - Access information about user’s private channels
    • Same as channels:read permission, but for private channels and conversations with multiple people.
  • users:read - Access your workspace’s profile information
    • In order to detect active users, we need to check user presence.
